# 🛒 SmartMart – AI Price Comparison System

SmartMart is a modern web application that helps users compare product prices across multiple e-commerce platforms and find the best deals instantly.

---

## 🚀 Features

* 🔍 Search any product (phones, laptops, fashion, etc.)
* 💰 Compare prices from different platforms (Amazon, Flipkart, etc.)
* 🏆 Highlights the best deal automatically
* 🖼️ Product cards with images, price, and store info
* 🤖 Smart chatbot assistant (helps users navigate and decide)
* ⚡ Fast and responsive UI
* 🎨 Modern UI with gradient + glassmorphism design

---

## 🧠 How It Works

1. User enters a product name
2. System fetches product data
3. Displays results in card format
4. Compares prices
5. Highlights lowest price as “Best Deal”

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js (Express)
* API: (Optional) SerpAPI / Mock Data
* Storage: LocalStorage (for user data)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/smartmart.git
```

2. Go to project folder

```bash
cd smartmart
```

3. Install dependencies

```bash
npm install
```

4. Run server

```bash
node server.js
```

5. Open browser

```
http://localhost:3000
```
